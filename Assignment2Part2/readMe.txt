- A project title
	ADT and TDD - XML Parser

- What the program does
	The program reads XML files formatted properly and produces 
	the correct output

- The date
	Nov 12, 2021

- The author
	Jihoon Oh, Eunji Lee, Jonghan Park and Bushra Osman

- How to run the program
	Parser.jar file is excutable file and trigger the program. The user 
	inputs the specific file name on the command line to parse a XML 
	file for getting a output from the program. The program reads each 
	correct tags and put it to stack and queue for going through the 
	program. The program stores error tags to errorQ and reads only 
	correct tags to run the program without errors.
